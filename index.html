Thoughts on Progressive Enhancement

Progressive enhancement basically means putting effort into the content of the web page first before anything else. The content is usually what makes a web page and then css and java script is like added decor to the content.

The content is more important than the decor and needs to be focused on first because users visit a website typically for the content and not the decor (css & javascript)
